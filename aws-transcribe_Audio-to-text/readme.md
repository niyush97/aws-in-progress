In this project you can learn about AWS Transcribe fundamentals along with creating and managing S3 Buckets using Terraform

#### AWS Transcribe.
- Creating Job
- Creating and applying a Customised Vocabulary
- Ceating and applying a Vocabulary filter
- Real-time Transcription

#### S3 (Simple Storage Service)
in this project the s3 bucket life-cycle will be automated using Terrafrom

```HCL
terraform init
terraform plan
terraform apply
```
---

### Cleanup of Resources.
- Delete the AWS Transcribe resources 
    - Jobs
    - Custom vocabulary 
    - Vocabulary Filter

- S3 buckets.
```HCL
terraform destroy
```
